üìù Usage Examples

Basic Usage

<script type="module" src="smart-card.js"></script> <smart-card title="My Card" subtitle="Card description"> <p>Your content here</p> </smart-card>

Advanced Usage

<smart-card type="product" title="Product Name" subtitle="Product description" image="image-url.jpg" href="https://example.com" badge="Sale" elevation="3" > <p>Product details and description</p> <div slot="footer"> <strong>$19.99</strong> </div> </smart-card>

Available Attributes

  • type: default, product, profile, article, minimal
  • title: Main heading text
  • subtitle: Secondary text
  • image: Image URL for the card
  • href: Makes the card clickable
  • badge: Small badge text in top-right corner
  • elevation: Shadow depth (0-5)
